Application Header / Masthead

Hide Header

Header Items are hidden but the header still displays and effects the page layout.

The show-app-logo property should be set to false and the header should have no children slots for this variation

Sign In Header

Plain white hexagon (no button click). This header variation should be used when the application header doesn't require the heaxagon to be a navigation button. This will usually occur on the sign-in page.

In this example, the hexagon is showing a single T "Transform". The T suggests that app, or current state of the app doesn't belong to a specific product.

Navigation Header

Gradient hexagon (with button click). The is-logo-clickable property determines the state of the hexagon logo. If it is true then it is rendered as a gradient and opend the tf-ng-side-drawer when clicked.
